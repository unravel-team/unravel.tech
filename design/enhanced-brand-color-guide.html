Updated Color Philosophy

New Direction: We now use green (#6CB33F) as the only accent color with black (#212529) for text and neutral elements. This creates a cleaner, more focused design with enhanced readability. Blue colors have been completely replaced with neutral grays and black.

Typography System

Titles: Barlow font family (weights 400-800) for all headings
Body Text: Inter font family (weights 300-700) for all regular text
Both fonts are loaded from Google Fonts for optimal performance.

Background CSS Variables

/* Enhanced Background System */
:root {
  /* Solid Backgrounds */
  --color-bg-primary: #F4F9FA;
  --color-bg-secondary: #F7FBFC;
  --color-bg-tertiary: #F0F7F9;
  --color-bg-surface: #FAFEFE;
  --color-bg-accent: #E4F4F7;
  
  /* Gradient Backgrounds */
  --bg-gradient-soft: linear-gradient(180deg, #F2F9FA 0%, #E8F5F7 100%);
  --bg-gradient-vibrant: linear-gradient(135deg, #E6F7FA 0%, #F0F8EB 100%);
  --bg-gradient-subtle: linear-gradient(145deg, #FAFEFE 0%, #F7FBFC 100%);
  
  /* Mesh Gradient */
  --bg-mesh: radial-gradient(at 40% 20%, #E6F7FA 0px, transparent 50%),
             radial-gradient(at 80% 0%, #F0F8EB 0px, transparent 50%),
             radial-gradient(at 10% 50%, #F7FBFC 0px, transparent 50%);
  
  /* Pattern Background */
  --bg-pattern: repeating-linear-gradient(45deg, 
    transparent, transparent 35px, 
    rgba(14, 159, 188, 0.03) 35px, 
    rgba(14, 159, 188, 0.03) 70px);
}
